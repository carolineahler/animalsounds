# animalsounds

<!-- badges: start -->

<!-- badges: end -->

The goal of animalsounds is to give the sound of an animal of your
choice. juhuu

## Installation

You can install the development version of animalsounds from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("carolineahler/animalsounds")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(animalsounds)
## basic example code
animal_sounds("cow", "muh")
#> [1] "The cow says muh!"
animal_sounds("giraffe")
#> [1] "The giraffe makes no sounds."
```
